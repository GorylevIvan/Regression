# Regression
!!!Свазана с работой Parsing а именно с файлами x_data.npy и y_data.npy!!!
## Обучение модели

Используется линейная модель **Ridge Regression**
(со стандартизацией признаков и логарифмированием таргета).

## Разбиение данных

- **60%** — train  
- **20%** — validation  
- **20%** — test  

## Метрики качества

Считаются следующие метрики:
- **MSE**
- **RMSE**
- **R²**

Также **опционально** обучается модель **CatBoostRegressor** для сравнения качества.

---

## Запуск приложения (обучение + предсказание)

### Первый запуск (обучение + предсказания)

```bash
python app.py x_data.npy y_data.npy
```
## Что произойдёт

- модель будет обучена

- модель сохранится в resources/model.joblib

- метрики сохранятся в resources/metrics.json

- в консоль выведутся:

    - метрики на validation и test

    - реальные и предсказанные зарплаты

## Повторный запуск (только предсказания)
```bash
python app.py x_data.npy
```

## Что произойдёт

- модель будет загружена из resources/model.joblib

- обучение будет пропущено

- в консоль выведутся:

    - метрики

    - редсказанные зарплаты

## Выводимые файлы

x_data.npy — массив признаков

y_data.npy — реальные зарплаты

resources/model.joblib — обученная модель

resources/metrics.json — метрики качества

## Используемые технологии

- Python

- NumPy

- Pandas

- scikit-learn

- Ridge Regression

- (опционально) CatBoost

## Примечания

- Проект ориентирован на линейные модели

- CatBoost используется только как эксперимент
